# PIK-R Website 🌿

A modern and responsive web-based platform developed for **PIK-R (Youth Information and Counseling Center)** as a digital medium for delivering educational content, organizational information, and youth-related activities in a more engaging and accessible way.

This project was built using **Python** for the backend implementation and focuses on creating an informative platform that supports youth empowerment, mental health awareness, self-development, and counseling services through technology.

## ✨ Key Features

* Modern and responsive user interface
* Interactive landing page
* Educational content related to youth and mental health
* Information portal for PIK-R activities and programs
* Simple and user-friendly navigation
* Lightweight and scalable web architecture

## 🛠️ Technologies Used

This project was developed using:

* **Python**
* **Flask**
* **HTML5**
* **CSS3**
* **JavaScript**

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash id="80q2j0"
git clone https://github.com/shanayabrr/PIK-R.git
```

### 2. Navigate to the Project Directory

```bash id="przfqx"
cd PIK-R
```

### 3. Install Dependencies

Make sure Python is installed on your computer, then install Flask:

```bash id="smkj50"
pip install flask
```

### 4. Run the Application

```bash id="v2h8cl"
python app.py
```
